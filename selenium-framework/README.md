# Selenium Automation Framework 🚀

## 📌 Project Overview
This project demonstrates web automation using Selenium with Python.  
It follows the Page Object Model (POM) design pattern for maintainability.

---

## 🛠 Tools & Technologies
- Selenium WebDriver
- Python
- pytest

---

## 🔍 Features
- Automated login functionality
- Implemented Page Object Model (POM)
- Used pytest for test execution
- Validated UI elements and responses
